Experiment 5: CSS Box Model Demonstration

This experiment shows how margin, padding, and border work in CSS.

Container: Margin = 30px, Padding = 20px, Border = 5px solid pink
Inner Box: Margin = 20px, Padding = 15px, Border = 3px solid green
Small Box: Margin = 10px, Padding = 10px, Border = 2px dashed orange Content inside the small box

Explanation:

  • Margin: Space outside the border, separates elements from each other.
  • Border: Surrounds the padding and content.
  • Padding: Space between content and border.
